# ElasticSearch Quick Started

### 集成方式
Spring Boot中集成Elasticsearch有4种方式：

- REST Client

- Jest

- Spring Data

- Spring Data Elasticsearch Repositories

本文用后面两种方式来分别连接并操作Elasticsearch
